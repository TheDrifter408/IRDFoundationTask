# IRD Foundation task:

## Stack used:
    1. NextJS
    2. Express (to create an API) along with Sequelize ORM
    3. Tailwind CSS

## To Run the project:
Clone this project:
```bash
git clone git@github.com:TheDrifter408/IRDFoundationTask.git
cd IRDFoundation
```
then run,
```bash
npm install
npm run express #to run the express server
npm run dev
```
